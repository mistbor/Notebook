### 简介

```
python 是一种高层次的结合了解释性、编译性、互动性和面向对象的脚本语言。
1. 解释性：在开发过程中没有了编译环节。
2. 交互性：可以在一个Python提示符直接互动执行写程序。
3. 面向对象：支持面向对象的风格或代码封装在对象的编程技术。
```

**优点**
```
易于学习
易于维护
易于阅读
一个广泛的标准库
互动模式
可移植
可扩展
数据库（python提供所有主要的商业数据库的接口）
GUI编程
可嵌入（python可嵌入到c/c++程序中，使程序的用户获得脚本化的能力)
```
